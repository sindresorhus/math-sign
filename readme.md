# math-sign

> ES2015 [`Math.sign()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/sign) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install --save math-sign
```


## Usage

```js
var mathSign = require('math-sign');

mathSign(5);
//=> 1

mathSign(-2);
//=> -1
```
